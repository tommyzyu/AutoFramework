## 如何使用



**安装依赖包**

1. 首先进入pip官网下载安装包，执行安装命令

   ```
   >python3  get-pip.py
   ```

2. 使用pip命令安装根目录中的"requirements"文件

   ```
   pip install -r requirements.txt
   ```

3. 进入项目根目录执行“run”脚本

   ```
   python run.py
   ```

**配置Pytest**

>
>
>addopts                          配置pytest的显示方式以及测试报告的显示方式
>
>testpaths                        配置测试用例路径
>
>python_files                   配置测试文件路径
>
>python_classes              配置测试类
>
>python_functions          配置测试函数
